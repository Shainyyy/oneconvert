How to Use the Image Quality Slider

The quality percentage controls how much data JPEG compression discards. At 100% quality, almost no data is discarded — the file remains large. At 60-70% quality, significant data is discarded — the file is much smaller but artefacts may appear around edges and fine details.

For most web images and email attachments, 75-85% quality provides the right balance. The live preview shows both versions simultaneously so you can compare before downloading.
