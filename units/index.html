Unit Conversion Reference Guide

The most frequently needed conversions: 1 kilometre = 0.621371 miles, 1 kilogram = 2.20462 pounds, 0°C = 32°F (formula: F = C × 9/5 + 32), 1 litre = 0.264172 US gallons, 1 inch = 2.54 centimetres, 1 acre = 4,046.86 square metres, 1 kilobyte = 1,024 bytes.

For web development unit conversions (px to rem, em, vh, vw), use the dedicated CSS Unit Converter. For programming and electronics (binary, hexadecimal, octal), use the Number Base Converter. For timestamp and epoch conversion, use the Unix Timestamp Converter.

Unit conversion is standardised internationally through the SI system (Système International d'Unités). The SI defines seven base units: the metre for length, kilogram for mass, second for time, ampere for electric current, kelvin for temperature, mole for amount of substance and candela for luminous intensity. All other units are derived from these. The imperial system used in the United States and some other countries uses older definitions that predate SI standardisation, which is why conversions involve non-round factors like 2.54 cm per inch rather than clean powers of 10.
