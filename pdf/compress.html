How Does PDF Compression Work?

PDF compression reduces file size by stripping embedded metadata — author names, keywords, creation tools, revision history and other hidden data that adds weight without adding value. OneConvert uses pdf-lib with object stream compression (equivalent to PDF/1.5 compression) to pack the remaining data more efficiently.

Results vary by PDF: documents heavy with metadata can shrink significantly, while already-optimised PDFs may see smaller reductions. Image-heavy PDFs require deeper compression (resampling images) which is a server-side operation. For those, use our compress tool to remove metadata first, then open in your PDF viewer to reduce image quality.
